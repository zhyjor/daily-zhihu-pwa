# daily-zhihu-pwa

一个用来测试pwa和ssr的vue知乎日报项目

## 分支
**master:** 普通的vue项目分支

**ssr:**	服务器渲染测试分支

**pwa：**	pwa测试分支

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run all tests
npm test
```
